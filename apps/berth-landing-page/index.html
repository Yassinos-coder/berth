Berth
Features GitHub
Open source · self-hosted

Deploy on infrastructure you own.

Berth is a self-hosted deployment panel for your VPS — the manageability of Railway or Render, on servers you control.

$ curl -fsSL https://berth.sh | sudo bash

One command installs the panel, agent, and database. Then open your browser and create an admin — that's it.

Your infrastructure

Runs on OVH, Hetzner, AWS — any Ubuntu or Amazon Linux box. No phone-home, no lock-in.

Deploy anything

Apps from GitHub, Docker images, databases, and buckets — every deployable is one ServiceSpec.

Secure by default

mTLS-authenticated agents, encrypted secrets at rest, httpOnly sessions, and RBAC teams.

Self-healing

Declarative reconciliation keeps your containers converged to desired state — idempotent, survives reboots.
